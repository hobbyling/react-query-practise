## Getting Started

### Prerequisites

node 需要使用 14 版以上

### Installation
1. Clone 專案
   ```sh
   git clone git@github.com:hobbyling/react-query-practise.git
   ```
2. 進入專案並安裝 NPM packages
   ```sh
   cd react-query-practise
   npm install
   ```
3. 本地端運行 json-server
   ```sh
   npm run json-server
   ```
4. 本地端運行
   ```sh
   npm run start
   ``` 
5. 開啟網頁：http://localhost:3000/
